# ShellScript

#### Manipulação de Arquivos 
Shell | Descrição
----- | ------------
  ">"   | arquivo de criação (sobrescrever)
  ">>" |  arquivo anexar ao arquivo( Não sobrescrevi, adiciona no final) 
  ">file 2>&1"   | tanto a saída quanto os erros sai para arquivamento
  "< file" | ler arquivo
